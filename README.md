# Ejercicio LineaPorLinea

**Escribe un programa que lea un archivo línea por línea y cuente cuántas líneas 
contienen una palabra específica.**

A usuario se pide escribir por teclado alguna palabra especifica, el programa lo recibe y guarda en un String.
Utilizando un BufferedReader y un bucle while, se va leer el fichero linea por linea buscando esta palabra y contando cuantas lineas lo tienen.
Al final, se muestra el resultado por la pantalla.
